Create a simple personal portfolio using HTML only.

Your portfolio should include:

- Your full name
- A profile picture
- A short biography
- Your hobbies
- Your skills
- Your goals as a frontend developer
- Your favorite quote



Create a file named style.css and connect it to your HTML page.

Style your portfolio by adding:

- A background color
- A readable font
- Different colors for headings and paragraphs
- Proper spacing using margin and padding
- A centered page title
- A border around your profile picture
- Rounded corners for the image
- Better spacing between sections
- A consistent color theme
- A neat and professional appearance

Your goal is to make the webpage visually appealing while keeping it simple.
